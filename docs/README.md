# Tài liệu Kỹ thuật Dự án Aloha

## Tổng quan

Dự án Aloha là một hệ thống quản lý với phần admin frontend (Next.js/TypeScript) và backend dự kiến sử dụng Fastify, Zod, tRPC, MongoDB.

Tài liệu này bao gồm:

- Kiến trúc hệ thống
- API documentation
- Thiết kế database
- Logic nghiệp vụ
- Hướng dẫn triển khai

## Cấu trúc Docs

- `architecture/`: Kiến trúc hệ thống
- `api/`: Tài liệu API
- `database/`: Thiết kế DB
- `business-logic/`: Logic nghiệp vụ
- `deployment/`: Triển khai
- `CONTRIBUTING.md`: Đóng góp
- `CHANGELOG.md`: Nhật ký thay đổi
