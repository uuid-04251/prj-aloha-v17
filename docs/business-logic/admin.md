# Logic Phần Admin

## Dashboard

- Hiển thị thống kê: số sản phẩm, danh mục, người dùng
- Charts cho doanh thu, đơn hàng

## Quản lý Sản phẩm

- CRUD operations
- Upload hình ảnh
- Phân loại theo danh mục
- Tìm kiếm và lọc

## Quản lý Danh mục

- Tạo/sửa/xóa danh mục
- Hierarchical categories (nếu cần)

## Quy trình

1. Admin đăng nhập
2. Xem dashboard
3. Quản lý data qua UI
4. API calls đến backend
